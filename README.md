# E-commerce Analytics Project

This project applies exploratory data analysis (EDA) techniques to a real-world e-commerce dataset to uncover insights into sales trends, customer behaviour, and revenue drivers.

## Key Features
- Data cleaning and preprocessing
- Feature engineering (revenue calculation)
- Exploratory data analysis (EDA)
- Customer and product performance analysis
- Business insights and recommendations

## Tools Used
- Python (Pandas, Matplotlib)
- Jupyter Notebook

## Key Insights
- Revenue is concentrated among a small number of customers
- Sales are driven by key product categories
- Customer satisfaction is generally high but linked to delivery outcomes

## Future Work
- Customer segmentation
- Demand forecasting
- Machine learning models
